Connect BadgeOS with Easy Digital Downloads

== Description ==

Gamify your Easy Digital Downloads store thanks to the powerful achievement plugin BadgeOS!

This plugin automatically connects BadgeOS with Easy Digital Downloads and also provides support for some of the most popular EDD extensions

= Available Triggers =

Easy Digital Downloads:

* New download: When an user publish a new download
* New purchase: When an user makes a new purchase (triggered 1 time on a purchase, independent of the number of downloads purchased)
* New download purchase: When an user purchases a download (triggered on every download purchased on a purchase)
* New free download purchase: When an user purchases a free download (triggered on every free download purchased on a purchase)
* New paid download purchase: When an user purchases a paid download (triggered on every paid download purchased on a purchase)

EDD FrontEnd Submissions:

* New download: When a vendors gets approved the first time a new download

EDD Wish Lists:

* New wish list: When an user creates a new wish list
* Add to wish list: When an user adds a download to a wish list

EDD Downloads Lists:

* Add to wishes list: When an user adds a download to wishes list
* Add to favorites list: When an user adds a download to favorites list
* Add to likes list: When an user adds a download to likes list
* Add to recommendations list: When an user adds a download to recommendations list

EDD Reviews:

* Review a download: When an user reviews a download
* Review a specific download: When an user reviews a specific download

EDD Download Pages:

* New download pages: When an user publish a new download page (also with support for EDD Download Pages FES integration)

EDD Social Discounts:

* Share download: When an user shares a download

This plugin requires [BadgeOS](http://wordpress.org/extend/plugins/badgeos/ "BadgeOS") and [Easy Digital Downloads](http://wordpress.org/extend/plugins/easy-digital-downloads/ "Easy Digital Downloads").

== Installation ==

= From WordPress backend =

1. Navigate to Plugins -> Add new.
2. Click the button "Upload Plugin" next to "Add plugins" title.
3. Upload the downloaded zip file and activate it.

= Direct upload =

1. Upload the downloaded zip file into your `wp-content/plugins/` folder.
2. Unzip the uploaded zip file.
3. Navigate to Plugins menu on your WordPress admin area.
4. Activate this plugin.
